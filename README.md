# ACM POTD 

This repository contains my daily solutions for the **ACM IGDTUW Problem of the Day (POTD)** challenge.

## Duration

22 March 2026 – 25 April 2026 (34 days)

## Objective

To build consistency in problem-solving by solving **1 DSA problem every day** and maintaining a public record of progress.

## Structure

Each day’s solution is uploaded as a Markdown file following this format:

```
Fullname_Date_Day-X.md
```

### Example:

```
Sadhika_22-03-26_Day-1.md
```

## What Each File Contains

* Problem statement (brief)
* Approach/logic
* Time & space complexity
* Code solution (C++)
* Screenshot of accepted submission

## Workflow

Solve → Document → Upload → Repeat

## Goal

Complete at least **30 out of 34 problems** to achieve consistency and strengthen core problem-solving skills.

## POTD Platform

Daily problems are provided by ACM IGDTUW:
https://acmigdtuw-potd.vercel.app/

## Note

This repository is maintained for learning, discipline, and tracking progress over time.
